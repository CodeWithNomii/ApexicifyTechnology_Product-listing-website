# Apexcify Technology – Product Listing Website

A modern and responsive product showcase website built using HTML, CSS, and JavaScript.  
This project presents futuristic technology solutions with interactive UI elements, animated sliders, and a premium dark-themed design.

---

## 🚀 Features

- Modern Dark UI Design
- Responsive Layout
- Interactive Product Slider
- Smooth Hover Animations
- Glassmorphism Effects
- About Section
- Stylish Footer
- Mobile Responsive Design
- Vanilla JavaScript Slider Functionality

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
